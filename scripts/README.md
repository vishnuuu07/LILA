# Discovery Scripts

This folder contains lightweight exploration scripts used to validate the dataset before production implementation.

- `inspect_schema.py` — opens sample parquet files and prints their schema
- `event_summary.py` — summarizes event counts and file counts
- `player_statistics.py` — summarizes player/bot distributions
- `match_statistics.py` — summarizes match counts and match-level distributions
- `coordinate_validation.py` — validates the coordinate projection formula and writes a plot
